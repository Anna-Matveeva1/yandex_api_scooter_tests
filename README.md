# Яндекс.Самокат - Автоматизация тестов API

# Описание проекта
Автоматизированные тесты для API сервиса Яндекс.Самокат.

# Тестовый сценарий
1. Клиент создает заказ
2. Проверяется, что по треку заказа можно получить данные о заказе

# Шаги автотеста:
1. Выполнить запрос на создание заказа
2. Сохранить номер трека заказа
3. Выполнить запрос на получение заказа по треку заказа
4. Проверить, что код ответа равен 200

# Технические примечания:
- Логи лежат в файле `error.log` в папке `/var/www/backend/logs`
